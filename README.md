# PlayFolio 🧩

## Highlights

- Single Page Application (SPA)
- Works with multiple accounts running and posting at the same time (Using session cookies)
- Allows for picture upload through drag and drop

## Overview

This project was designed as a great resource for game developers to showcase their games and games they worked on in a nice organized place. The site focuses on professionals working in the video games industry and uses posts in a card layout in the application to present the projects. Once you post a new project you can give it a description as well as your role, the current status of the game as well as additional info on the company. You can follow users as well to create a small network of individuals with similar interests.

This was the final term project for my advanced website and design class (COMP 3540).
